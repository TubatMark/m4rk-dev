# Portfolio Website

A modern, fully-featured developer portfolio built with Next.js 16, React 19, and Convex. Features a beautiful animated frontend and a complete admin dashboard for managing all content.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI Library:** React 19
- **Database:** Convex (real-time backend)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Theme:** next-themes (dark/light mode)
- **Icons:** Lucide React
- **Language:** TypeScript

## Features

### Frontend
- Responsive, animated landing page
- Hero section with social links
- About section with skills, technologies, and stats
- Projects showcase with filtering
- Contact form with real-time submission
- Dark/light theme toggle
- Smooth scroll navigation

### Admin Dashboard
- Secure authentication system
- Manage all portfolio content:
  - **Projects** - Add, edit, delete portfolio projects
  - **Messages** - View and manage contact form submissions
  - **Hero Section** - Edit name, headline, status badge, CTAs
  - **About Section** - Update about title and description
  - **Skills** - Manage skill cards with icons
  - **Technologies** - Edit technology tags
  - **Stats** - Update achievement statistics
  - **Social Links** - Manage social media links
  - **Experience** - Add work experience timeline
  - **Contact Section** - Edit contact info
  - **Site Settings** - SEO, branding, footer content

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm
- Convex account

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd mark-portfolio
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Set up Convex**
   ```bash
   npx convex dev
   ```
   This will prompt you to log in and create a new project.

4. **Start the development server**
   ```bash
   pnpm dev
   ```

5. **Create your admin account**
   - Visit `http://localhost:3000/admin-portal/setup`
   - Fill in your name, email, and password
   - Click "Create Admin Account"
   - **Important:** Delete the setup page after creating your account:
     ```bash
     rm -rf "app/(admin)/admin-portal/setup"
     ```

6. **Access the admin dashboard**
   - Navigate to `/admin-portal`
   - Log in with your credentials

## Environment Variables

Create a `.env.local` file in the root directory:

```env
# Convex (auto-generated by `npx convex dev`)
CONVEX_DEPLOYMENT=dev:your-deployment-name
NEXT_PUBLIC_CONVEX_URL=https://your-deployment.convex.cloud
```

## Database Schema

The Convex database includes the following tables:

| Table | Purpose |
|-------|---------|
| `projects` | Portfolio projects |
| `messages` | Contact form submissions |
| `adminUsers` | Admin accounts |
| `sessions` | Authentication sessions |
| `siteSettings` | SEO and branding |
| `heroSection` | Hero section content |
| `aboutSection` | About section content |
| `skills` | Skill cards |
| `technologies` | Technology tags |
| `stats` | Achievement statistics |
| `socialLinks` | Social media links |
| `experience` | Work experience |
| `contactSection` | Contact info |

## Customization

### Adding New Icons

Icons are mapped in each component. To add new icons:

1. Import from `lucide-react`
2. Add to the `iconMap` object in the component

### Theming

Edit `app/globals.css` to customize:
- Color variables (CSS custom properties)
- Font settings
- Animation keyframes

### Adding New Sections

1. Create component in `components/sections/`
2. Add Convex schema and functions if needed
3. Add admin page in `app/(admin)/admin-portal/dashboard/`
4. Update sidebar menu in `components/admin/sidebar.tsx`

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import project in Vercel
3. Add environment variables
4. Deploy

### Convex Production

```bash
npx convex deploy
```

## Security Notes

- Admin routes are protected by session-based authentication
- Passwords are hashed before storage
- Sessions expire after 7 days
- The setup page should be deleted after initial admin creation
- Never commit `.env.local` to version control
